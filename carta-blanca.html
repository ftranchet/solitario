Cómo jugar

  1. Arrastrá una carta hasta donde va, o tocala para tomarla y después tocá el destino.
  2. Arriba a la izquierda hay 4 pozos libres: cada uno guarda una sola carta para usarla después.
  3. En las columnas se baja alternando colores (una roja, una negra).
  4. Para mandar una carta a las pilas de arriba (derecha), tocala dos veces.

Objetivo: armar las cuatro pilas de arriba a la derecha desde el As hasta el Rey, una por palo.

Podés mover varias cartas juntas si forman una escalera (bajando y alternando color). Cuantos más pozos y columnas vacías tengas, más cartas podés mover de una vez.

Si te trabás, tocá 💡 Pista abajo (de nuevo para ver otra jugada). Cuando todo quede ordenado aparece ✓ Autocompletar para terminar la partida sola.
