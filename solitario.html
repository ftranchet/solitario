Cómo jugar

  1. Tocá el mazo (arriba a la izquierda) para dar vuelta cartas.
  2. Arrastrá una carta hasta donde va, o tocala para tomarla y después tocá el destino.
  3. Para mandar una carta a las pilas de arriba, tocala dos veces.

Objetivo: ordenar las cuatro pilas de arriba desde el As hasta el Rey, una por palo. En las columnas se baja alternando colores (una roja, una negra).

Si te trabás, tocá 💡 Pista abajo (tocala de nuevo para ver otra jugada). Cuando ya no queden cartas tapadas aparece ✓ Autocompletar para terminar la partida sola.
